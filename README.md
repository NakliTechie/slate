# Slate

IrfanView for the browser. No install. No upload. Any OS.

---

## Features (v1.0)

- **Edit:** crop, resize, redact (black-out), blur, rotate, flip, straighten, brightness/contrast, sharpen, text, arrow
- **Viewer:** zoom, pan, fit modes, fullscreen, ruler overlay
- **Folder album:** open a folder, navigate with arrow keys, thumbnail strip
- **Open:** drag-drop, folder picker, file picker, paste from clipboard
- **Save:** download (default), overwrite original, save as sidecar `-edited`
- **Formats in:** JPG, PNG, WebP, GIF (first frame), AVIF, SVG, BMP
- **Formats out:** PNG, JPG, WebP
- **EXIF:** displayed on open, stripped on save by default
- **Undo/redo:** 50 steps

## Privacy

Nothing leaves your browser. Slate makes zero network requests after the page loads. No accounts, no telemetry, no uploads.

## Browser support

| Browser | Version | Notes |
|---|---|---|
| Chrome / Edge | 111+ | Full support including folder access |
| Firefox | 113+ | File picker + download (no folder write-back) |
| Safari | 16.4+ | File picker + download (no folder write-back) |
